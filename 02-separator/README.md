# 02 — Separator

**Separator** يفصل بين أجزاء الرسالة ويخليها أسهل للقراءة.

## متى أستخدمه؟

- فصل الأقسام داخل نفس الرسالة.
- تقسيم معلومات طويلة.

## مثال سريع

```js
await channel.send({
  components: [
    {
      type: 1, // Container
      components: [
        { type: 10, content: 'قسم أول: معلومات الحساب.' },
        { type: 14 }, // Separator
        { type: 10, content: 'قسم ثاني: إعدادات الخصوصية.' }
      ]
    }
  ]
});
```

## نصائح

- لا تكثر الفواصل بدون داعي.
- الأفضل استخدامه بين مجموعات مختلفة من المعلومات.
