# أخطاء شائعة وحلولها

هذه صفحة سريعة لأكثر الأخطاء اللي تصير عند استخدام Discord Components V2.

## 1) الرسالة لا تظهر المكوّنات

**السبب المحتمل:** ما استخدمت Container أو الصيغة غلط.

**الحل:** تأكد أن الرسالة تحتوي على `components` وبداخلها `Container`.

```js
await channel.send({
  components: [
    { type: 1, components: [{ type: 10, content: 'نص واضح' }] }
  ]
});
```

## 2) الزر ما يشتغل

**السبب المحتمل:** `custom_id` مفقود أو التفاعل غير مستمع.

**الحل:** أضف `custom_id` وتأكد من listener للأحداث.

## 3) الـ Select Menu ما يعرض خيارات

**السبب المحتمل:** الخيارات ناقصة `label` أو `value`.

**الحل:** عرّف `label` و `value` لكل خيار.

## 4) الصور ما تظهر في Media Gallery

**السبب المحتمل:** رابط الصورة غير مباشر أو غير آمن.

**الحل:** استخدم روابط مباشرة (`https://`) لصورة فعلية.

## 5) رسائل طويلة ومزعجة

**السبب المحتمل:** نصوص كثيرة بدون فواصل.

**الحل:** استخدم Separator وقلّل الحشو.
